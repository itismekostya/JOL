# JOL Catalog Content

```text
image filename
NAME — $PRICE
Short description
Image mode: ordinary | fullscreen
Message to buy:
```

Если у одного объекта несколько фото, просто повторяем блок с тем же названием, ценой и описанием, но с другим `image filename`.
